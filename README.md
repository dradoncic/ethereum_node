## Getting Started

1. Clone the repository:
   ```bash
   git clone https://github.com/<your-username>/ethnode.git
   cd ethnode

2. Set up your Erigon node:
   ```bash
   cd erigon
   make erigon
   ./build/bin/erigon

3. Use the scripts in the scripts/ folder to interact with the node
